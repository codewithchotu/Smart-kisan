# 🌾 Smart Kisan

**Smart Kisan helps farmers test farming decisions before investing, reducing risk using data.**

Smart Kisan is a simple web application built to support farmers in making better decisions using basic inputs like soil, weather, and crop choice.

The idea is straightforward:  
👉 Instead of taking risks in real life, farmers should be able to test their decisions first.

---

## 🚀 Why we built this

In many places, farming decisions are still based on experience and assumptions. While that works sometimes, changing weather patterns and soil conditions make it risky.

Some common problems we observed:

- Farmers don’t always know their soil condition clearly  
- Weather is unpredictable  
- Fertilizers are often used without proper guidance  
- Choosing the wrong crop can lead to financial loss  
- There’s no way to “try before you plant”  

We wanted to build something that reduces this uncertainty.

---

## 💡 What Smart Kisan does

The app allows users to enter a few simple inputs:

- Soil type  
- pH value  
- Rainfall level  
- Crop  

Based on this, the system provides:

- Estimated yield  
- Cost and profit  
- Better crop suggestions  
- Fertilizer recommendations  
- Basic weather insights  

---

## 🔥 What makes it different

### 🧪 Strategy Comparison (What-if Simulator)

Instead of showing just one result, the app compares multiple strategies:

- Low investment  
- Balanced  
- High investment  

This helps users understand which approach works best before making real decisions.

---

### 🧠 Decision Support

The system doesn’t just show numbers — it suggests the best option based on profit and risk.

---

### 📊 Learning from past data

We store previous inputs and use them to generate insights like:

> “In similar conditions, wheat gave better results.”

This makes the system smarter over time.

---

## 🧩 Tech Stack

We kept the stack simple and practical:

- **Frontend:** React  
- **Backend:** Node.js + Express  
- **Database:** MongoDB  

---

## 🔗 APIs Used

- **Open-Meteo** → weather data  
- **data.gov.in** → crop market prices  
- **Nominatim (OpenStreetMap)** → location detection  
- **Twilio / WhatsApp API** → notifications
